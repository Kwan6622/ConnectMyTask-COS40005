# Changelog

All notable changes to the ConnectMyTask Mobile app will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-02-06

### Added
- Initial React Native mobile app setup with Expo
- Task management core features (CRUD operations)
- User authentication store and state management
- Task priority system (Low, Medium, High)
- Task completion tracking
- Due date management
- Reusable UI components (Button, Input, TaskCard, Badge)
- TypeScript support with strict mode
- Zustand state management integration
- Axios API client setup
- Task utility functions for formatting and priority handling
- Jest testing setup
- ESLint and Prettier configuration
- Development and project documentation

### Features
- ✅ Create tasks with title and description
- ✅ Set task priority levels
- ✅ Mark tasks as complete/incomplete
- ✅ Delete tasks
- ✅ View task list with statistics
- ✅ Authentication state management
- ✅ Responsive mobile-first design

### Documentation
- README.md with setup instructions
- DEVELOPMENT.md with coding guidelines
- Comprehensive JSDoc comments in services
